# 🛍️ Retail Customer Segmentation Analysis

## 📌 Project Overview

This project applies RFM (Recency, Frequency, Monetary) analysis to segment retail customers based on their purchasing behavior. Using Python for data preparation and Power BI for visualization, the project identifies high-value customers, loyal customers, and customers at risk of churn, enabling businesses to make data-driven marketing decisions.

---

## 🎯 Project Objectives

- Clean and preprocess retail transaction data.
- Perform RFM (Recency, Frequency, Monetary) analysis.
- Segment customers based on purchasing behavior.
- Build an interactive Power BI dashboard.
- Generate business insights and recommendations for customer retention.

## 💡 Key Business Insights

- Loyal customers contribute a significant portion of total revenue.
- High-value customers purchase more frequently and spend more per transaction.
- Some customer segments show signs of inactivity and require re-engagement.
- RFM analysis effectively identifies customers for targeted marketing campaigns.

---

## 📌 Business Recommendations

- Reward loyal customers through exclusive offers and loyalty programs.
- Re-engage inactive customers with personalized promotions.
- Focus marketing efforts on high-value customer segments.
- Monitor customer behavior regularly to reduce churn.
- Use RFM segmentation to improve campaign targeting and return on investment.

---

## 📈 Project Outcome

The project demonstrates how customer segmentation using RFM analysis can help businesses identify valuable customers, improve customer retention, optimize marketing campaigns, and increase revenue through data-driven decision-making.
